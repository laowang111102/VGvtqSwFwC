# Python Django 图书管理系统 python660

## 项目概述

本系统是一个基于Python Django框架开发的图书管理系统，旨在提供便捷的图书信息管理功能。

## 技术栈

- 后端：Python Django
- 数据库：MySQL

## 功能模块

- 用户管理：登录、注册、登出、修改密码
- 图书管理：添加图书、管理作者、管理出版社
- 信息查询：查找出版社信息、查看图书信息
- 安全性：密码加密、信息唯一性判断

## 系统角色

- 系统管理员：负责整个系统的管理和维护
- 普通用户：可以查看图书信息，进行个人账户管理

## 运行环境

- 操作系统：兼容主流操作系统
- Python版本：建议使用Python 3.x
- 数据库：MySQL

## 部署步骤

1. 环境准备：确保Python、Django、MySQL已正确安装
2. 配置数据库：根据实际情况配置MySQL数据库
3. 导入数据库：导入系统所需的初始数据
4. 运行服务器：通过Django命令运行开发服务器

## 目录结构

```
project/
│
├── apps/               # 系统应用目录
├── db/                 # 数据库文件目录
├── media/              # 媒体文件目录
├── static/             # 静态文件目录
├── templates/          # 模板文件目录
│
├── manage.py           # 管理脚本
├── requirements.txt    # 依赖列表
└── ...
```

- 基于成熟稳定的Python Django框架开发，易于维护和扩展
- 提供完善的功能模块，满足图书管理的日常需求
- 密码加密机制，确保用户信息安全
- 灵活的信息查询功能，方便用户快速定位所需图书信息
- 清晰的目录结构和详细的部署文档，便于系统部署和二次开发
